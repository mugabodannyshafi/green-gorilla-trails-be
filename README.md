# green-gorilla-trails-be

A professional, dynamic, mobile-first content management system for tourism operations specializing in Rwanda gorilla trekking and East Africa safaris.


## Installation

```bash
pnpm install
```

## Running the app

```bash
# development
pnpm run start

# watch mode
pnpm run start:dev

# production mode
pnpm run start:prod
```

## Test

```bash
# unit tests
pnpm run test

# e2e tests
pnpm run test:e2e

# test coverage
pnpm run test:cov
```
